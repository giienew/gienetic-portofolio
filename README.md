# Ayogi Akbar — Developer Portfolio

Modern, responsive portfolio website built with iOS 2026 design language.

## Features

- **3 Theme Modes** — Dark, Light, Midnight Blue (switchable in real-time)
- **iOS-style UI** — Glassmorphism, bottom tab bar, sheet modals
- **Fully Responsive** — Mobile-first, supports all devices
- **Smooth Animations** — Intersection Observer based, performant
- **Project Showcase** — Filter by category, detail modals with gallery
- **Contact Form** — Direct WhatsApp integration
- **REST API** — [exsalapi.my.id](https://exsalapi.my.id)

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Inter + JetBrains Mono (Google Fonts)
- Font Awesome 6.5
- No frameworks — pure, fast, lightweight

Hosted via GitHub Pages.
